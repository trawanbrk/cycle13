# Cycle OS

Открытая операционная карта года: 13 циклов, 52 недели, 4 фазы.

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Тестовый режим даты

```
http://localhost:3000/?date=2026-06-22
http://localhost:3000/?date=2026-12-28
http://localhost:3000/?date=2027-01-04
```
